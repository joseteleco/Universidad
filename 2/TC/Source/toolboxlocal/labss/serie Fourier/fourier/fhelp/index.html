Installation :

Installation: Extract the files from the archive, making sure to retain the original folder structure. If you wish, add the directory to the MATLABPATH.

To Start: If you have added the installation directory to the MATLABPATH when you installed it, you can just run the dconvdemo.m function and the GUI will start. If you have not added the installation directory to the MATLABPATH, first change to the directory in which the files were extracted and then run the dconvdemo.m function. The program will ask you if it is okay to add the installation directory to the path for the duration of the current Matlab session. Answer yes.

Quick guide :

1. Start the program by typing 'fourierdemo' in the MATLAB command window. The program appears.

2. Then choose the signal type from the pop up menu. On the left, the waveform window will give the actual representation of the signal. The magnitude and phase spectrum will be diplayed in the next windows.

3. Choose the number of fourier coefficients you want to use. You can do this either from the edit box or by using the slider. As you do this you will notice the change in magnitude and phase spectrum. Also the signal reconstructed from the fourier coefficients is displayed in the waveform window in red.

4. For help click the 'help' button.

5. To exit, click the 'close' button
